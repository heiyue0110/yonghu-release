# 永狐 1.4.14 更新公告

发布日期：2026-07-17

1.4.14 修复配装低槽点击和当前位置读取两条实际使用链路。配装页右下资源读数不再遮挡槽位；EVE SSO 令牌过期时会自动刷新后重试位置，短时网络异常也不会把生产预警永久切成手动模式。

## 配装交互

- 修复右下角 CPU、能量栅格等资源读数层级过高，遮挡底部第一个低槽点击的问题。
- 资源读数改为不接收鼠标事件，低槽可以正常选中、切换状态和装卸模块。
- 使用毒蜥级实装页面逐个点击三个低槽验证，槽位均能响应，既有配装计算与资源显示保持不变。

## EVE SSO 与当前位置

- 生产预警读取当前位置遇到 HTTP 401 或过期 Token 时，会自动刷新 EVE SSO 令牌并重试一次。
- 星图、角色页、诊断页和角色资料预热统一使用同一位置刷新逻辑，不再由不同入口各自漏报过期登录。
- 未登录、缺少位置权限、网络/代理异常和刷新失败会显示对应原因，避免一律误报成普通频道权限问题。

## 生产预警稳定性

- 临时无法读取当前位置时，如果已有上次生产星系，本轮继续使用该星系扫描附近风险。
- 自动跟随保持开启，下一轮会重新读取当前位置；不会因为一次 ESI 或网络抖动永久改成手动星系。
- 没有可用历史位置时仍允许手动输入星系，不会自动开启监控、安装或其他高风险操作。

## 更新说明

- 软件会自动检查公开更新清单，但下载、安装和重启前仍需用户明确确认。
- 取消更新后不会继续下载、安装或重启。
- 安装器和自动更新清单使用配套 Tauri 签名；签名校验失败会直接终止更新。
- 可直接覆盖安装，现有配置和缓存会继续保留。
